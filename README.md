# .gitdigital-zkThe Root Identity, Trust & Governance Stack for the GitDigital zk Organization

The GitDigital zk Organization is a zero‑knowledge–native digital civilization built around privacy‑preserving identity, compliance, governance, and community‑aligned infrastructure.

This is the source of truth for the GitDigital zk ecosystem.

📦 Included Stacks

This repo currently distributes two core bundles:

1. gitdigital-full-stack.zip
A complete, end‑to‑end environment for:

- zk Identity  
- zk Compliance  
- zk Governance  
- Off‑chain verification daemons  
- SDKs (Rust + TypeScript)  
- Example integrations  
- CI/CD pipelines  
- Developer tooling  
- Deployment scripts  

This is the canonical full-stack bundle for GitDigital zk.

---

2. gitdigital-trust-stack.zip
A lightweight, trust‑layer–focused bundle containing:

- Identity masks  
- Trust scoring engines  
- Off‑chain verification  
- Authority badge logic  
- Minimal SDK footprint  

This is ideal for integrators, auditors, and partners who only need the trust layer.
